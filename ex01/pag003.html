Trabalhando com listas

Listas ordenadas

Utilizada quando a ordem dos elementos é relevante. Representada pela tag <ol>. Exempo:

  1. Acordar;
  2. Tomar café;
  3. Escovar os dentes.

Listas não ordenadas

Utilizada quando a ordem dos elementos não é relevante. Representada pela tag <ul>. Exempo:

  • Carne;
  • Ovos;
  • Manteiga.
